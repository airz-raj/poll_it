# Poll-It: Interactive Election Education Assistant

A stunning, highly interactive web application designed to educate citizens about the democratic process, timelines, and electoral systems worldwide.

## Features
- **Interactive Timeline:** Follow the journey from voter registration to counting day.
- **Voting Simulator:** A step-by-step gamified voting experience.
- **Global Comparison Arena:** Compare election metrics across different countries visually.
- **Knowledge Quiz:** Test your civic knowledge across multiple categories.
- **AI Assistant:** Get instant, contextual answers to your election queries.
- **Accessibility:** Full suite of accessibility features including high contrast and dyslexia-friendly modes.
